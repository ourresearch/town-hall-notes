# OpenAlex Town Hall Notes

Quarterly update notes for the [OpenAlex](https://openalex.org) community town hall and Community Advisory Board (CAB) meetings.

Each quarter has a directory with two files: a **retro** (what we shipped) and a **roadmap** (what's next). The retros are generated with the help of an AI agent (Claude Code) using our open-source repos, blog, and commit history as sources. The exact prompt is included at the top of each retro — so anyone can reproduce or adapt the process.
