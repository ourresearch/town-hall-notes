# OpenAlex Town Hall Notes

Quarterly update notes for the [OpenAlex](https://openalex.org) community town hall and Community Advisory Board (CAB) meetings.

Each file covers one quarter and includes a summary of what we shipped, data quality improvements, and community highlights. The notes are generated with the help of an AI agent (Claude Code) using our open-source repos, blog, and commit history as sources. The exact prompt is included at the top of each file — so anyone can reproduce or adapt the process.
